#README.md 
##Stopwatch Web Application

A modern and interactive stopwatch web application built using **HTML, CSS, and JavaScript**, featuring a fascinated UI/UX design and accurate time tracking with lap functionality.

---

##Task Overview

The goal of this project is to create a user-friendly stopwatch web application that enables users to measure time intervals accurately. The application supports essential stopwatch operations such as start, pause, reset, and lap tracking, all wrapped in an attractive and responsive interface.

---

## Features

-  Start the stopwatch
-  Pause the running timer
-  Reset the stopwatch
-  Record and display lap times
-  High-precision time measurement
-  Modern glassmorphism-inspired UI
-  Fully responsive design

---

## Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling, animations, and UI/UX design  
- **JavaScript (ES6)** – Stopwatch logic and interactivity
